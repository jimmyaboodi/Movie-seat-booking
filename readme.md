# Movie Seat Booking

Detta är min inlämningsuppgift för Movie Seat Booking.

## Loggbok

**Dag 1:** Satte upp grundprojektet med HTML och CSS från start-repot. Läste på om hur `json-server` fungerar och skapade min lokala databas (`db.json`) med några filmer och priser.

**Dag 2:** Delade upp Javascript-koden i moduler för att uppfylla kraven. Skapade en `Movie`-klass i en egen fil och skrev logiken för att hämta filmerna via `fetch` från mitt lokala API och bygga upp rullgardinsmenyn dynamiskt.

**Dag 3:** Byggde klick-logiken för stolarna. Såg till att stolar byter färg till "Selected" när man klickar, och lade in en spärr så att stolar märkta med "Occupied" ignoreras. Skrev en funktion som dynamiskt räknar ut totalpriset baserat på vald film och antal stolar. Felsökte och testade koden.
